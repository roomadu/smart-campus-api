Smart Campus Sensor and Room Management API

Module: 5COSC022W - Client-Server Architectures

Base URL: /api/v1

Available Endpoints

Discovery

Method URL Description
GET /api/v1 API discovery - returns metadata and resource links

Rooms

Method URL Description
GET /api/v1/rooms Get all rooms
POST /api/v1/rooms Create a new room
GET /api/v1/rooms/{roomId} Get a specific room by ID
DELETE /api/v1/rooms/{roomId} Delete a room (blocked if sensors are still assigned)

Sensors

Method URL Description
GET /api/v1/sensors Get all sensors (use ?type= to filter by type)
POST /api/v1/sensors Register a new sensor (roomId must exist)
GET /api/v1/sensors/{sensorId} Get a specific sensor by ID
DELETE /api/v1/sensors/{sensorId} Remove a sensor

Sensor Readings

Method URL Description
GET /api/v1/sensors/{sensorId}/readings Get all readings for a sensor
POST /api/v1/sensors/{sensorId}/readings Add a new reading (blocked if sensor is in MAINTENANCE)

Error Responses

Status Scenario
409 Conflict Deleting a room that still has sensors assigned
422 Unprocessable Entity Creating a sensor with a roomId that does not exist
403 Forbidden Posting a reading to a sensor in MAINTENANCE status
500 Internal Server Error Any unexpected server error (no stack trace is exposed)

Use Postman or curl to test the API endpoints listed above.
